How to use this prototype
You can try these in chat
1. Asking a clarifying question about the document
Try prompt: what is this document about
Output: You should see response without edit.
2. Some big changes in document level
Try prompt: rewrite the document for legal filing
Output: To see doc level changes, with loading interaction.
3. Asking super app to edit a section in document level
Try prompt: make the intro section concise
Output: See the change happening in document.
4. Assume someone clicked on "leadership" section and it is locked, and you are asking for a change in that section.
Try prompt: update the leadership team
Output: Super app reply with edit queued tag.
